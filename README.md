# Graceful Tadpoles

Tadpoles are graphs that consist of a cycle and a path that are joined together. It was shown that all tadpoles are graceful, in the sense that they allow a graceful labeling of the vertices. 

Sadly, part of these results are only available in Chinese. This repository contains my inofficial translation of the paper

Guo Wenfu. The Gracefulness of Graph B(m,n). Journal of Inner Mongolia Normal University (Natural Science Edition), 1994(2): 25-29.

together with python code that implements the constructions presented in the paper.
